Variables

StringToBooleanMap: TStringToBooleanMap;

Mappings from strings to booleans.

This is used by the StringToBoolean function and can be modified. The default values are:

  • '1', 't', 'true', 'on', 'y', 'yes', 'j', 'ja' -> True

  • '0', 'f', 'false', 'off', 'n', 'no', 'nein' -> False
